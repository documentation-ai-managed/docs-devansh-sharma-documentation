## Theme and Branding

Tailor the visual appearance starting with your brand color `#3B82F6`.

### Set Brand Color

Follow these steps to update your theme:

<Steps>
  <Step title="Access Settings" icon="settings">
    Navigate to **Admin > Themes** in the dashboard.
  </Step>
  <Step title="Enter Brand Color" icon="palette">
    Input `#3B82F6` in the **Brand Color** field.
  </Step>
  <Step title="Preview and Save" icon="save">
    Preview changes and click **Apply**.
  </Step>
</Steps>

For advanced customization, edit the configuration file directly:

<CodeGroup tabs="JSON,YAML">
```json
{
  "theme": {
    "brandColor": "#3B82F6",
    "logo": "https://your-domain.com/logo.svg",
    "fontFamily": "Inter, sans-serif"
  }
}
```

```yaml
theme:
  brandColor: "#3B82F6"
  logo: https://your-domain.com/logo.svg
  fontFamily: Inter, sans-serif
```
</CodeGroup>

## User Permissions and Access Control

Define roles to manage who can edit, view, or admin your documentation.

| Role       | View | Edit | Admin | Description                          |
|------------|------|------|-------|--------------------------------------|
| Viewer     | ✅   | ❌   | ❌    | Read-only access to all pages.      |
| Editor     | ✅   | ✅   | ❌    | Edit content but no settings.       |
| Admin      | ✅   | ✅   | ✅    | Full control including users.       |

<Tabs>
  <Tab title="UI Method" icon="mouse-pointer">
    Go to **Users > Permissions**. Assign roles via dropdown.
  </Tab>
  <Tab title="API Method" icon="api">
    Use the permissions endpoint:

    ```bash
    curl -X POST https://api.devansh-docs.com/v1/permissions \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -d '{"userId": "user123", "role": "editor"}'
    ```
  </Tab>
</Tabs>

<Expandable title="Advanced Permission Rules" default-open="false">
  Create custom rules with conditions like IP restrictions or time-based access.
</Expandable>

## Export and Import Options

Export your space as Markdown or JSON for backups. Import to migrate content.

<Steps>
  <Step title="Export" icon="download">
    Select **Tools > Export**. Choose format and download.
  </Step>
  <Step title="Import" icon="upload">
    Upload file in **Tools > Import**. Review and confirm.
  </Step>
</Steps>
